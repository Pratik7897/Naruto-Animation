# Naruto & Sasuke Hand Tracking 🌀⚡

A premium interactive web experience using **MediaPipe Hands** to trigger Naruto and Sasuke visual effects (Jutsu) based on hand gestures!

## ✨ Features
- **Real-time Hand Tracking**: Powered by MediaPipe.
- **Dynamic VFX**: Choose between Naruto's Rasengan and Sasuke's Chidori.
- **Responsive Design**: High-performance canvas overlay.
- **Premium Aesthetics**: Dark mode with glowing skeletons.

## 🛠️ Tech Stack
- **HTML5 & CSS3**
- **JavaScript (Vanilla)**
- **MediaPipe Hands API**
- **MediaPipe Camera Utils**

## 🚀 How it Works
1. **Open** `index.html` in your browser.
2. **Grant Camera Access** when prompted.
3. **Gestures**:
   - **Open Left Hand**: Triggers Naruto's Rasengan VFX.
   - **Open Right Hand**: Triggers Sasuke's Chidori VFX.
   - Closing your hand will fade out the effects.

## 📁 Repository Structure
- `index.html`: The core application code.
- `naruto.mp4`: Naruto Jutsu VFX asset.
- `sasuke.mp4`: Sasuke Jutsu VFX asset.
- `.gitignore`: Standard git exclusion patterns.
